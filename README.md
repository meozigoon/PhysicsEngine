# **PhysicsEngine**

F.R.I.D.A.Y.님의 글을 보며 만들었습니다. (https://pang2h.tistory.com/376?category=888812)

Visual Studio에서 LNK2019 오류가 발생할 경우, 프로젝트 속성 → 링커 → 시스템 → 하위 시스템 '콘솔'을 '창'으로 바꾸면 해결

PhysicsEngine.exe 파일은 완성본 실행 파일입니다.

## <조작 방법>

Left: ←, A

Right: →, D

Jump: ↑, Space
